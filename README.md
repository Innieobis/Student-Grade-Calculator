# Student-Grade-Calculator
# Python Calculator Project

This is a simple **Python Calculator** project for SEN 201 assignment.  
It demonstrates basic operations like addition, subtraction, multiplication, and division.  

---

## Features
- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers (handles division by zero)

## How to Use
1. Open `calculator.py` in any Python IDE or online editor.
2. Run the program.
3. Choose an operation (1-Add, 2-Subtract, 3-Multiply, 4-Divide).
4. Enter two numbers.
5. Get the result!
